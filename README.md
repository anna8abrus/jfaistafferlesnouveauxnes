# PPM Image Generator in C++

This program generates an image in the PPM (Portable PixMap) format using random colors for each pixel. The generated image is saved as a 250x250-pixel file named `image.ppm`. The PPM format is a simple image format that stores pixel data as text, allowing easy inspection of image data. This `README.md` explains the code in detail.

## Features

1. **PPM Image Generation**: The program creates an image in the PPM format using random RGB values for each pixel.
2. **Fixed Dimensions**: The image size is 250x250 pixels.
3. **Output**: The image is saved in a file named `image.ppm`.
